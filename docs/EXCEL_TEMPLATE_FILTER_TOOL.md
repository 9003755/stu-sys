# Excel 模板筛列工具使用说明

## 功能说明

该工具用于处理“字段很多、但只需要保留模板列”的 Excel 文件。

用户上传一个源 Excel 后，工具会：

1. 读取系统内置模板 `stuIm.xlsx`
2. 识别模板第一行中的目标列
3. 自动忽略源表中模板没有的列
4. 按模板列顺序重组数据
5. 生成一个新的 Excel 文件供下载

## 适用场景

- 学员报名信息来源不统一，列很多
- 只想保留模板中要求的字段
- 需要把结果整理成与系统模板一致的列顺序

## 使用入口

启动前端项目后，打开：

```text
/tools/excel-template-filter
```

例如本地开发环境通常是：

```text
http://localhost:5173/tools/excel-template-filter
```

## 使用步骤

1. 打开工具页
2. 点击“选择 Excel 文件”
3. 上传包含学员报名信息的源表
4. 等待系统自动分析列匹配情况
5. 点击“生成新 Excel”
6. 下载整理后的结果文件

## 处理规则

- 固定使用系统内置模板：`public/stuIm.xlsx`
- 固定读取模板第一个工作表
- 固定读取源 Excel 的第一个工作表
- 固定使用第一行作为表头
- 表头按文本匹配
- 模板中没有的列会被自动忽略
- 模板中存在但源表缺失的列，会在结果文件中保留为空列

## 输出文件

输出文件名格式：

```text
源文件名_按模板整理.xlsx
```

## 注意事项

- 当前仅支持 `.xlsx`
- 如果源表第一行不是表头，需要先手工整理后再导入
- 如果源表列名与模板列名差异太大，系统会识别为“缺失列”
- 若模板后续改了列名或列顺序，工具输出也会随之变化
